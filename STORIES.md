# User stories

- [Som en användare vill jag se en kontextuell bild och text "Läslistan" ovanpå webbsidan så att jag kan veta på vilken webbplats är jag.](src/features/general_page_layout.feature)
- [Som en användare vill jag se böcker i Katalog så att jag kan välja bocker.](src/features/books_visible_in_catalog.feature)
- [Som en användare vill jag märka en eller flera böcker i Katalog så att jag kan spara mina favorita böcker.](src/features/favorite_books_list.feature)
- [Som en användare vill jag kunna ta bort en favoritmarkering i Katalog så att jag kan hantera min lista när jag tittar igenom böcker.](src/features/favorite_books_list.feature)
- [Som en användare vill jag se mina favorit-märkta böcker på en separat sida "Mina böcker" så att jag kan titta igenom mina favorita böcker lista.](src/features/favorite_books_list.feature)
- [Som en användare vill jag se en text "När du valt, kommer dina favoritböcker att visas här." på sida "Mina böcker" när jag har inga märkta böcker så att jag kan förstå att märkta böcker visas på denna sida.](src/features/favorite_books_list.feature)
- [Som en användare vill jag kunna lägga till böcker (titel och författare) i Katalog från separat sida "Lägg till bok" så att jag kan fylla på Katalogen.](src/features/adding_books.feature)
- [Som en användare vill jag lägga till en bok bara när jag skrev både titel och författare så att jag kan ha fullständig information i Katalogen.](src/features/adding_books.feature)
- [Som en användare vill jag landa på Katalog sida och växla mellan Katalog, Lägg till bok och Mina bocker sidor så att jag kan navigera hur jag vill på webbplatsen.](src/features/navigation.feature)
- [Som en användare vill jag se text "Läslistan" och bok-ikon vid sidans titel högst upp i webbläsarfönstret så att jag kan skilja webbplatsen från andra flikar i webbläsaren.](src/features/general_page_layout.feature)
