# Testautomatisering-Handelsakademin-Examination-Valeriia-Ermakova

This repository contains files for exam for Testautomatisering och testverktyg course.

The web-site which is tested: <https://tap-vt25-testverktyg.github.io/exam--reading-list/>

Features which are tested:

- General layout
- Navigation
- Catalog view
- Adding new books to Catalog
- Marking and unmarking favorite books
- Adding books to My books list

How to start the project:

0. Clone this repository to your work folder: `git clone https://github.com/valeriiaermakova/Testautomatisering-Handelsakademin-Examination-Valeriia-Ermakova.git`
1. Run console command (recommended in a separate virtual environment): `pip install -r requirements.txt`
2. OR install manually Python, Pytest, Behave, Playwright
3. Navigate to [src](./src/) folder
4. Run console command: `behave`
